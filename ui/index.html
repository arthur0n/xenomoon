Harvest a transcript

Paste a raw video transcript. The framework saves it to the game's transcripts/ folder and runs transcript-researcher on it.

Get assets

Each request below needs a file — pick a local file or paste its path, and choose a Place. PNGs land in textures/, GLBs in models/ — under the game's assets/ (default) or the external x-shared-assets/ library (free-library example assets kept out of the game tree). Then godot-dev imports and wires it. Fill as many as you like — the panel stays open until you close it.

Draw a level

Pick a tool and drag on the grid — wall · door · window · item · room (empty stays floor). Item and Room paint the chosen id (same id = same item / one room). Export saves the grid and hands it to the level-designer, who asks the level name + scene details, then has godot-dev build it.
