# Lighting in Godot — transcript digest

**Source** — `lighting.md` (the raw, now in `transcripts/archive/lighting.md`) (GDQuest-style "how to light a scene in Godot 4.4" video; tavern indoor + medieval-town outdoor examples, day→night swap).
**Why harvested** — starting Phase 5 (Light & environment) of `docs/roadmap/first_game.md`. Gate: sun shadows visible under the player (jump landing readable); no blown highlights. The transcript is the source-push input for the `godot-lighting-environment` skill that phase calls for.

**Stack reminder for the mapping** — Godot 4.3+, Forward+, reversed-Z, orthographic fixed-angle camera, 3D pixel art rendered in a SubViewport (nearest filter, AA off), GDScript-only, composition over inheritance. The current build is a _graybox blockout_ with one DirectionalLight3D + one WorldEnvironment/ProceduralSky already present in `levels/blockout_01.tscn`.
