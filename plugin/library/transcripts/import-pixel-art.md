**Verification note (the recent bug):** the video's "set sampling to Nearest" is the material-inspector path; the equivalent the build got wrong is `StandardMaterial3D.texture_filter`. Correct value is `1` (TEXTURE_FILTER_NEAREST); `3` is `TEXTURE_FILTER_NEAREST_WITH_MIPMAPS`, which _looks_ nearest but re-introduces mipmap blur at distance/glancing angles — exactly the regression the skill must lock out. The video does NOT cover the `.import` sidecar (`mipmaps/generate=false`, `compress/mode=0`) at all — that half of the skill comes from elsewhere.
