# Design docs

Each file here is one small, agreed slice of work produced by the **game-designer** agent (Opus). A doc must be implementable by **godot-dev** in one task and verifiable with **godot-verify**.

Workflow: idea → game-designer (interviews, cuts scope, writes doc) → godot-dev (implements) → godot-verify (checks) → human look.

No code is designed or written here beyond the agreed slice — parked ideas live in each doc's "Later" section.
